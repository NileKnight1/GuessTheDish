# Guess The Dish

## How to play?
- You can play it in your browser.
- The link: https://nileknight.itch.io/guessthedish

A game made written in RenPy.

The game idea is simple:
- You get dishes images and you have to guess what it is.
- You have a score and health points.
- You control the difficulty level and number of questions.
